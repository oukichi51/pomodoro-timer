# 休憩タイマー

ブラウザ上で動作する、シンプルな **集中／休憩タイマー** です。  
集中モードと休憩モードを切り替えながら、ポモドーロ風の作業サイクルをサポートします。

---

## 機能

- **モード**
  - 集中モード
  - 休憩モード
- **各モードの時間設定**
  - 集中モード：初期値 25 分
  - 休憩モード：初期値 5 分
  - どちらもユーザーが「分」単位で任意の値に変更可能
- **画面表示**
  - 現在時刻（HH:MM:SS）
  - タイマーの残り時間（タイマーの監視時間）
  - 現在のモード（集中モード / 休憩モード）
  - 操作用ボタン
    - スタート / 停止ボタン
    - リセットボタン
    - モード切替ボタン
- **タイマー終了時の挙動**
  - ブラウザ上に `alert` で通知を表示  
    （「集中モードが終了しました…」等）
  - OK を押すと自動的にモードが切り替わり、
    次のモードの時間でタイマーが準備される

---

## 見た目（カラーモード）

現在のモードに応じて、画面全体の色が変わります。

- **集中モード**
  - ページ背景を **暖色系（オレンジ寄り）** に変更
- **休憩モード**
  - ページ背景を **寒色系（青・水色寄り）** に変更

これにより、ぱっと見で「今は集中なのか休憩なのか」が分かりやすくなっています。

---

## 使い方

1. `index.html` をブラウザで開く
2. 必要に応じて、画面上部の入力欄から  
   - 集中モード（分）
   - 休憩モード（分）  
   を設定する
3. 「スタート」ボタンを押すと、現在のモードの時間でカウントダウン開始
4. 「停止」ボタンでもう一度押すと一時停止
5. 「リセット」ボタンで、現在モードの時間設定に応じた残り時間に戻す
6. 「モード切替」ボタンで、集中モード ↔ 休憩モードを手動で切り替え
7. タイマーが 0 になると `alert` 表示 → OK を押すと自動で次のモードに切替

## ファイル構成

- `index.html`  
  画面の構造（現在時刻、残り時間表示、ボタン類など）を定義
- `style.css`  
  レイアウトとスタイル、モードごとの背景色切り替えなど
- `script.js`  
  タイマーのロジック・モード管理・時間設定・アラート表示など

ビルドツールやバックエンドは不要で、  
静的ファイルをそのままブラウザで開くだけで動作します。

---

## 実装メモ

- **技術スタック**
  - HTML / CSS / 素の JavaScript（フレームワーク不使用）
- **現在時刻**
  - `setInterval` で 1 秒ごとに更新
- **タイマー**
  - `setInterval` による 1 秒単位のカウントダウン
  - 二重起動を防ぐため、タイマーIDを管理
- **モードごとの時間**
  - 集中／休憩それぞれの分数入力を読み込み、秒に変換して内部管理
- **色切り替え**
  - `body` に `focus-mode` / `break-mode` クラスを付け替えることで、
    CSS 側で背景色を変更
